# Contributing

Thanks for your interest in contributing! We welcome pull requests, bug reports, and feature suggestions.

## How to Contribute

1. **Fork the repository**.
2. **Create a new branch** for your feature or bugfix:
   ```bash
   git checkout -b feature/my-new-feature
   ```
3. **Make your changes** and commit them with clear messages.
4. **Push to your fork**:
   ```bash
   git push origin feature/my-new-feature
   ```
5. **Open a Pull Request** against the `main` branch.

## Code Style

- We use **Prettier** for code formatting.
- We use **ESLint** for linting.
- Please ensure your code passes `npm run build` before submitting.

## Reporting Bugs

Please open an issue on GitHub with a detailed description of the bug, steps to reproduce, and your environment (browser, OS).
